## Configuração do Ambiente
1. Clone o repositório:
   ```bash
   git clone https://github.com/nseguros/hackathon-challenge.git
   cd hackathon-challenge
   git checkout adolfo_quende
   ```
2. Instale as dependências:
   ```bash
   flutter pub get
   ```
3. Rodar
   ```bash
   flutter run
   ```
4. Build

   ```bash
   flutter build apk
   ```
